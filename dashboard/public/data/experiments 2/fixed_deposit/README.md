# Fixed Deposit: Isolating Skill Effect

**Plot**: 2x2 panel — skill sigma(t) (top-left), effective wager m(t) (top-right), cumulative profit (bottom-left), tail-average sigma vs m bar chart (bottom-right). All deposits b_i=1, 200 rounds.

**Results**:
- Skill correctly differentiates agents when deposits are equal.
- Better forecasters get higher sigma, larger wagers, and positive cumulative profit.
- Isolates the pure skill channel — no confounding from deposit variation.
