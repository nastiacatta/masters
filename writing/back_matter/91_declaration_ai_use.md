# Declaration of generative-AI use

Per the Dyson School of Design Engineering AI policy (DESE70002
handbook, Procedures page), this notice records the generative-AI
tools used during the preparation of this report, and the purpose
for which they were used. The handbook requires that this notice
appear immediately after the list of references.

## Tools used

- **Anthropic Claude (Sonnet and Opus families)** — used as a pair
  programming assistant inside the Kiro IDE for:
  - drafting, editing, and stress-testing prose in the report and
    presentation materials;
  - scaffolding Python code for experiments, plotting, and the
    dashboard;
  - reviewing derivations against the primary literature
    (Lambert et al. 2008; Ranjan and Gneiting 2010; Kuleshov et
    al. 2018).

- **GitHub Copilot** — in-editor single-line and short-block code
  completions during software development.

## Use and attribution

All ideas, judgements, and final prose in this report are the
author's. Outputs from the tools above were treated as third-party
suggestions; no output was used verbatim in the report without
review, verification against primary sources, and editing into the
author's own voice. Where a tool surfaced a specific factual claim
(e.g. a scoring-rule property, a published result, a library API
detail), the claim was cross-checked against the cited source
before inclusion.

No AI tool was used to generate experimental results, numerical
outputs, or figures. All numbers in the report are traceable to a
committed artefact in the repository.

No AI tool was granted access to the oral-examination materials
beyond their use as a drafting aid for the presentation script.
