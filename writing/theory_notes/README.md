# Theory notes

One note per primary theoretical source. Each note is structured the
same way:

- **Citation** — full entry + DOI / arXiv.
- **One-line take** — what the thesis claims that depends on this
  paper.
- **Relevant result** — the specific theorem or formula we use, with
  the paper's numbering preserved.
- **Where we use it** — chapters in `writing/*.md` that cite this.
- **Open question** — anything unresolved about our use of the result.

These notes are for paraphrasing and citation, not for re-deriving the
papers. When writing a chapter, pull a claim from the relevant note.
